# 第一章 东亚气旋与反气旋

本章讲的是影响中国天气的两类大尺度系统：气旋（低压、逆时针旋转、上升运动、阴雨天气）和反气旋（高压、顺时针旋转、下沉运动、晴好天气）。考试重点在于它们的源地、分类、移动路径，以及三类气旋（蒙古、江淮、黄河）各自的形成机制。

## 1.1 基本概念

- **气旋**：在同一高度上，中心气压低于四周的大尺度涡旋。==北半球气流逆时针旋转==，南半球相反。伴随上升运动，多阴雨天气。辐合
- **反气旋**：在同一高度上，中心气压高于四周的大尺度涡旋。北半球气流顺时针旋转，南半球相反。伴随下沉运动，多晴好天气。辐散

  > [!TIP]
  > 北逆南顺

## 1.2 东亚气旋的源地和分类

东亚气旋分两大类，与东亚南北两支锋区一一对应：

| 类别   | 纬度范围      | 具体名称                | 对应锋区   |
| ---- | --------- | ------------------- | ------ |
| 北方气旋 | 45°N-55°N | 蒙古气旋、东北气旋、黄河气旋、黄海气旋 | 东亚北支锋区 |
| 南方气旋 | 25°N-35°N | 江淮气旋、东海气旋           | 东亚南支锋区 |

**分布规律**：东部多，西部少；平原多，盆地高原少。我国110°E以西和20°N以南很少产生锋面气旋。

**高空锋区具备锋面气旋生成条件**：斜压性大、冷暖空气交汇处温度平流强、涡度平流明显。

## 1.3 气旋的移动

- **移向**：沿500hPa或700hPa引导气流移动。大多数由西南→东北或由西→东移动。
- **移速**：平均30-40公里/小时。春季快，夏季慢；初生阶段快，锢囚消亡阶段慢。与高空气流速度呈正比。

## 1.4 东亚反气旋的活动地区与路径

**活动地区**：蒙古西部-河套地区。冬季可南下到华南沿海，夏季在河套以北（40°N以北）。

==**移动路径**==（与进入我国的冷空气路径一致，共4条）：

| 路径                                                            | 走向                          |
| ------------------------------------------------------------- | --------------------------- |
| I 西北路径                                                        | 亚洲西北部→西西伯利亚→蒙古→我国           |
| II 偏北路径                                                       | 亚洲北部→西西伯利亚→蒙古→河套；或→东西伯利亚→东北 |
| III 偏西路径                                                      | 亚洲西部→巴湖（巴尔喀什湖）→新疆→河套        |
| IV 蒙古路径                                                       | 蒙古→我国                       |

## 1.5 蒙古气旋

**季节特征**：一年四季均有，春秋季最多。冬季冷高压控制少气旋，夏季北支锋区北移蒙古暖空气占优势。

**地形作用**：蒙古气旋生成的源地在蒙古中部和东部——**背风坡低压**。

地形对气旋/反气旋的影响：

- **迎风坡**：上升运动随高度减小，气柱压缩产生水平辐散→气旋性涡度减小→有利反气旋发展
- **背风坡**：下沉运动随高度减小，气柱拉长产生水平辐合→气旋性涡度增加→有利气旋发展

东亚气旋源地均在高大山脉东侧（蒙古气旋、黄河气旋、江淮气旋）。

**地面形势（形成过程）—— 3类**：

| 类型               | 形成机制                               | 说明            |
| ---------------- | ---------------------------------- | ------------- |
| 1. ==暖区新生类==（最多） | 背风坡暖区内低压形成→冷空气越山冷锋进入低压→高空槽前暖平流暖锋锋生 | 最典型，占比最高      |
| 2. ==冷锋进入倒槽==    | 暖性倒槽向北发展低压形成→冷锋越山进入倒槽→高空槽前暖平流暖锋锋生  | 倒槽来自中亚或新疆     |
| 3. ==蒙古副气旋类==    | 两股冷空气嵌形作用形成低压→暖锋+冷锋→与先前锋面气旋相连形成气旋族 | 北股冷空气从萨彦岭以北进入 |

3种地面的差别在于低压形成的方式不同。

**高空形势（500hPa）—— 3型**：

| 类型             | 特征                                               |
| -------------- | ------------------------------------------------ |
| 1. ==两槽一脊型==   | 乌山、东亚为槽，中亚为脊。地面蒙古有冷高压，24小时后移走，气旋发展。等高线疏散结构导致地面减压 |
| 2. ==槽前纬向气流型== | 西西伯利亚疏散槽，温度场稍落后于高度场。槽前辐散最大在贝湖附近。气旋产生在准葛尔盆地或蒙古地区  |
| 3. ==北脊南槽型==   | 两槽间蒙古为弱脊，南方阿尔泰山有一槽。槽脊连线西部疏散区域移到蒙古地区形成地面减压        |

**三型共同特点**：

1. 蒙古西部为疏散槽——正涡度平流强
2. 温度槽落后于高度槽——为发展槽
3. 锋区和冷暖平流均较强

**形成过程总结**：有利地形（背风坡）和发展疏散槽前的暖平流→蒙古中部先出现低压（或暖性倒槽）→叠加高空槽前正涡度平流使低压发展→槽前暖平流促使暖锋锋生→冷锋越山进入低压→锋面气旋形成。

**天气**：==大风==（沙尘暴、吹雪）、==降温==（寒潮、霜冻、降雪）、==降水少==（北方气候干燥，倒槽型降水较多）。

## 1.6 江淮气旋

**季节特征**：==四季均有，春季和初夏最多==。6月长江中下游最多，7月淮河最多，湘赣秋季以后为0。自西南向东北移动。

**生成地区**（三个）：

- ==长江中下游==（62%）—— 西起宜昌东至长江口，向东北东经东海到日本海
- 淮河流域（28%）—— 淮河沿岸，从海州入海经黄海到日本海
- 湘赣地区（10%）—— 全年春夏稍多，秋季几乎没有，大多从长江口入海

**地面形势 —— 2类**：

| 类型                      | 形成机制                                                                           |
| ----------------------- | ------------------------------------------------------------------------------ |
| 1. ==静止锋上的波动==（江淮梅雨锋波动） | 江淮流域有近似东西向的准静止锋→上空有低槽移来→高空槽前正涡度平流减压，低压形成→槽前暖平流使锋面北移（东段转为暖锋）→槽后冷平流使锋面南移（西段转为冷锋） |
| 2. ==倒槽锋生气旋==           | 地面变性高压东移入海→高空南支锋区西南气流将暖空气向北输送→地面减压形成倒槽并东伸→倒槽东部暖式风切变暖锋锋生→西风带发展小槽东移冷锋进入倒槽        |

**高空形势 —— 2型**：

- ==槽脊型==（500hPa、700hPa  ：高纬(45°N以北)长波槽脊稳定，无强冷空气南下；中纬(30-40°N)有发展短波槽东移形成锋面气旋。
- ==切变线型==（700hPa、夏季）：长江中下游江淮切变线，西南涡沿切变线东移，低涡下方诱生江淮气旋。

**天气**：

- ==江淮气旋是造成江淮地区暴雨的重要天气系统==（强降水、夏季暴雨系统）
- 迅速发展的江淮气旋伴有较强大风：暖锋前偏东大风、暖区偏南大风、冷锋后偏北大风
- 气旋东部出现平流雾、低云

## 1.7 黄河气旋

**季节特征**：四季均有，夏季最多。

**天气形势 —— 2类**：

| 类型             | 季节  | 形成机制                                                    |
| -------------- | --- | ------------------------------------------------------- |
| 1. 倒槽型（夏季型）    | 夏季  | 西南暖倒槽发展伸向河套、华北→低压形成→副高后部暖湿气流暖锋锋生→冷锋随西风带发展槽东移进入倒槽→锋面气旋形成 |
| 2. 静止锋波动型（冬季型） | 冬季  | 高空有东西向锋区，小槽自新疆移到河套北部→导致河套准静止锋上产生气旋（类同江淮气旋），一般发展不大       |

**天气**：

- ==冬季型==：**降水量小，产生大风、沙尘天气**
- ==夏季型==：**降水量大，暴雨天气系统**（华北地区暴雨重要天气形势）

### 联想与辨析（重点）

1. **地形作用的独特性**：只有**蒙古气旋**的生成与地形（背风坡）有直接且明确的关系，另外两个气旋主要与锋面和高空槽有关。
2. **降水的关键差异**：

   - **蒙古气旋**：**“干”**，主打大风、沙尘，降水少。

   - **江淮气旋**：**“湿”**，是长江流域暴雨的“代言人”。

   - **黄河气旋**：**“变”**，冬天像蒙古气旋（干），夏天像江淮气旋（湿），是华北暴雨的“罪魁祸首”之一。
3. **时间的“最”字诀**：

   - **蒙古气旋**：**春秋**最多。

   - **江淮气旋**：**春季和初夏**最多（6月）。

   - **黄河气旋**：**夏季**最多。

## 1.8 本章速记（考试必背）

1. 北方气旋对应北支锋区（45-55°N），南方气旋对应南支锋区（25-35°N）
2. 气旋沿500/700hPa引导气流移动，多为西南→东北方向
3. 东亚反气旋有4条路径：西北、偏北、偏西、蒙古
4. 蒙古气旋3种地面形势（暖区新生类最多、冷锋进入倒槽、副气旋类），3种高空形势（两槽一脊、槽前纬向气流、北脊南槽）
5. 江淮气旋2种地面形势（静止锋波动、倒槽锋生），2种高空形势（槽脊型、切变线型）
6. 黄河气旋2种形势（夏季倒槽型、冬季静止锋波动型）
7. 地形作用：背风坡辐合有利气旋发展，迎风坡辐散有利反气旋发展

# 第二章 寒潮天气过程

本章讲的是大规模强冷空气活动过程——寒潮。考试重点包括寒潮的分级标准、冷空气源地和路径、主要天气系统（极涡、极地高压、冷高压、冷锋），以及中期和短期天气过程的分类比较。

## 2.1 寒潮的定义与分级标准

**定义**：_寒潮（cold wave）是一种大规模的强冷空气活动过程。主要特点是剧烈降温和大风，有时伴有雨、雪、雨凇或霜冻。_

**冷涌**（cold surge）：当冷空气推进到南海时称为冷涌。

### 单站寒潮标准

| 等级       | 降温幅度要求                    | 日最低气温要求 | 速记         |
| -------- | ------------------------- | ------- | ---------- |
| **寒潮**   | 24h≥8℃，或48h≥10℃，或72h≥12℃  | 下降到4℃以下 | 8、10、12；4  |
| **强寒潮**  | 24h≥10℃，或48h≥12℃，或72h≥14℃ | 下降到2℃以下 | 10、12、14；2 |
| **特强寒潮** | 24h≥12℃，或48h≥14℃，或72h≥16℃ | 下降到0℃以下 | 12、14、16；0 |

注：日最低气温也可以用日平均气温替代。

### 区域性寒潮分类（新标准）

| 分类       | 标准                               |
| -------- | -------------------------------- |
| **北方寒潮** | 40%以上气象站满足单站寒潮标准                 |
| 北方强寒潮    | 50%以上气象站满足，且其中40%以上达强寒潮标准        |
| 北方特强寒潮   | 60%以上满足，或50%以上满足强寒潮且40%以上达特强寒潮   |
| **南方寒潮** | 30%以上气象站满足单站寒潮标准                 |
| 南方强寒潮    | 40%以上满足，且其中30%以上达强寒潮，或20%以上达特强寒潮 |
| **全国寒潮** | 满足北方寒潮 + 南方寒潮                    |
| 全国强寒潮    | 满足北方寒潮 + 南方强寒潮                   |
| 全国特强寒潮   | 北方特强寒潮 + 南方50%以上满足寒潮             |

### 旧标准补充

旧标准以过程降温和温度负距平划分：过程降温为日平均气温最高值与最低值之差；温度负距平为最低值与多年旬平均值之差。全国性寒潮需同时影响华北和长江两个区。

**出现时间**：9月-次年5月，春季3月和秋季9-11月最频繁。

## 2.2 冷空气的源地和路径

### 三个源地

| 源地                 | 路径方向 | 特点                                      |
| ------------------ | ---- | --------------------------------------- |
| I 新地岛以西的寒冷洋面（巴伦支海） | 西北路径 | 出现次数最多，达寒潮强度也最多                         |
| II 新地岛以东的寒冷洋面（喀拉海） | 北方路径 | 出现次数少，但气温低，可达寒潮强度                       |
| III 冰岛以南洋面         | 西方路径 | 出现次数多，温度不很低，一般达不到寒潮强度；与其他源地冷空气汇合后可达寒潮强度 |

### 寒潮关键区

70°-90°E，43°-65°N（西伯利亚中部地区）—— 95%的冷空气经过此地并积累加强。

### 从关键区入侵我国的四条路径

> [!TIP]
> 这里与三个冷空气源地区别开

| 路径       | 走向                   | 天气特点                      |
| -------- | -------------------- | ------------------------- |
| 1. 西北路   | 关键区→蒙古→河套→长江中下游及江南   | 以偏北大风和降温为主，到江南后可能有雨雪      |
| 2. 东路    | 关键区→蒙古→华北北部→东北→入海    | 渤海、黄海、黄河下游出现东北大风，华北东北连阴雨雪 |
| 3. 西路    | 关键区→新疆→青海→西南→华南      | 降温幅度不太大，若与南支锋区叠加可有明显降温    |
| 4. 东路加西路 | 华北冷空气回流南下与青海冷空气汇合→江淮 | 大范围雨雪，合并南下出现大风和明显降温       |

## 2.3 寒潮主要天气系统

### （一）极涡

**定义**：极地附近的冷性大尺度涡旋，是寒潮冷空气的源地。深厚系统，越到高层越强。

**出现频数最多地区**：北美-极区-亚洲北部（西半球多于东半球），平均持续天数超过5天。

**移动路径三种类型**：

1. 经向性移动——极涡中心从一个半球经极地摆到另一半球
2. 纬向性移动——多发生在欧亚大陆高纬地区
3. 转游性移动——在极区亚洲部分和喀拉海-新地岛一带，方向多向西

**四种环流型**：

| 类型      | 特征                  |
| ------- | ------------------- |
| 绕极型     | 仅一个中心，基本在极区（80°N以北） |
| **偏心型** | 一个中心偏于东/西半球（80°N以南） |
| **偶极型** | 北美与亚洲北部各一个低压中心      |
| 多极型     | 分裂为多个低压中心           |

**对寒潮的作用**：我国中等以上强度大范围持续低温与极涡分裂为偶极型有关。_**只有极涡分裂变形，才有利于寒潮冷空气的形成。**_

> [!NOTE]
>
> - 记住**极涡是深厚系统**，同时根据口诀==暖低冷高薄，冷低暖高厚==来推测其他系统是否是深厚/浅薄系统
> - 暖性低压和冷性高压是浅薄系统，冷性低压和暖性高压是深厚系统
> - 温度对气柱密度&高度的影响

### （二）极地高压

**定义**：中高纬度阻塞高压进入极区的深厚暖性高压。条件：500hPa有完整反气旋环流、有单独暖中心配合、暖高压主体在70°N以北、维持3天以上。

**形成方式**：

1. **经向发展型**——暖温度脊向北伸入极区，中纬高度脊经向发展形成
2. **低涡切断型**——长波冷槽加深南下切断为冷低压，冷槽前高脊发展伸入极区

**作用**：

- 极地高压发展建立→极涡从极区南下（分裂变形，有利寒潮冷空气形成）
- 极地高压衰退南垮→冷空气向南爆发，引发寒潮

### （三）寒潮地面冷高压

==中心强度 ≥ 1050hPa==。多数属于热力不对称系统：_前部强冷平流，后部暖平流_，中心区温度平流趋于零。中心移动路径可作为冷空气的移动路径。

### （四）寒潮冷锋

在寒潮地面高压前缘都有一条强度较强的冷锋。锋区结构上宽下窄。定锋参考850hPa最明显（锋区窄）。强度要求：

- 850hPa — 20℃/5纬距
- 700hPa — 16℃/5纬距

## 2.4 寒潮中期天气过程（2-3周）

寒潮爆发前的大尺度环流背景。

### 三大类中期过程

| 类型              | 占比      | 演变三阶段                                                |
| --------------- | ------- | ---------------------------------------------------- |
| 1. ==**倒Ω流型**== | 70%-80% | 初始（两洋暖脊发展→极涡分裂为偶极型）→ 酝酿（东亚倒Ω流型建立）→ 爆发（横槽转竖/南压→冷空气爆发） |
| 2. **极涡偏心型**    | —       | 初始（极涡偏于东半球）→ 酝酿（东亚倒Ω流型建立）→ 爆发（乌山脊强烈发展东移→冷空气爆发）       |
| 3. **大型槽脊东移型**  | —       | 北半球长波槽顺序东移，槽后冷空气南下引发寒潮。移动性长波系统，无突然爆发过程               |

### 倒Ω流型详细演变

**==初始==阶段**：大西洋暖高脊和阿拉斯加暖高脊向极地发展（或两脊打通）→极涡分为东西两半球各一个中心（偶极型）→东半球倒Ω流型形成。

**==酝酿==阶段**：东半球倒Ω流型向亚洲地区收缩→乌拉尔山和鄂霍次克海建立暖性高压脊→亚洲极涡加强并南压→东亚倒Ω流型形成。

**==爆发==阶段**：中纬长波槽脊急速发展东移或以横槽形式（横槽转竖或横槽南压）→引导冷空气向南爆发→东亚大槽重建，寒潮结束。

### 两个大洋中暖脊的作用

关键系统：==**大西洋暖高脊**==和==**阿拉斯加暖高脊**==（_太平洋暖高脊_）它们对东亚两个暖高脊（乌拉尔山高压脊、鄂霍次克海暖高压脊）的形成起主要作用。

**==乌拉尔山暖高脊==的形成**：

1. **补充型**：乌山浅脊与大西洋暖脊东伸的正变高叠加而发展加强
2. **结合型**：乌山浅脊与阿拉斯加暖脊切断出的西移极地高压叠加而发展加强
3. **叠加型**：乌山浅脊与黑海高脊叠加而发展加强

**鄂霍次克海暖高脊的形成**：阿拉斯加高压脊西退到鄂霍次克海发展而成，或阿拉斯加高压脊分裂出小高压西移与鄂海弱脊合并而成。

综上，_**寒潮中期预报的关键系统是两个大洋上的暖性高压脊**_。

## 2.5 寒潮短期天气过程（3-7天）

### 五大类短期过程

#### （一）小槽发展型

| 项目      | 内容                         |
| ------- | -------------------------- |
| 冷空气源地   | 新地岛以西的洋面（巴伦支海）             |
| 冷空气路径   | 西北路径（乌拉尔山→西西伯利亚→蒙古→我国）     |
| 关键区入侵路径 | 西北路为主（全国性寒潮），东路次之（北方寒潮）    |
| 环流特征    | 乌山长波高压脊发展建立，脊前西北气流引导冷空气南下  |
| 爆发方式    | 脊前不稳定小槽边东移边发展，在贝湖地区发展成长波大槽 |
| 东亚大槽    | **重建**（北半球长波一次调整）          |

#### （二）低槽东移型（西来槽型）

| 项目      | 内容                             |
| ------- | ------------------------------ |
| 冷空气源地   | 冰岛以南洋面（欧洲西海岸）                  |
| 冷空气路径   | 西方路径+西北路径→合并叠加                 |
| 关键区入侵路径 | 东路为主（北方类寒潮），西北路次之→全国类寒潮        |
| 环流特征    | 欧洲弱脊东移到中亚发展成长波脊                |
| 爆发方式    | 西北或北方新冷空气叠加合并→西来低槽在贝湖强烈发展成长波大槽 |
| 东亚大槽    | **重建**（北半球长波一次调整）              |
| 有利因素    | "赶槽"、南支槽配合                     |

#### （三）横槽型（最重要，爆发最猛烈）

**横槽转竖型**：

| 项目    | 内容                              |
| ----- | ------------------------------- |
| 冷空气源地 | 新地岛以东的洋面（喀拉海或新西伯利亚海）            |
| 冷空气路径 | 北方路径（泰梅尔半岛→亚洲北部→蒙古→我国）          |
| 关键区入侵 | 可不进入关键区，自北向南影响                  |
| 环流特征  | 西伯利亚阻塞高压与横槽稳定建立，高纬呈"北高南低"形势     |
| 爆发方式  | 阻高西退（或崩溃），横槽转竖，冷空气爆发性南下（一天影响全国） |
| 东亚大槽  | **重建**（北半球长波一次调整）               |

**横槽转竖的预报要点**：

1. 横槽后由冷平流转为暖平流，冷平流侵入横槽前
2. 阻高脊线由东北-西南向转为西北-东南向，横槽后由东北风转为西北风
3. 横槽前有阶梯槽或等高线疏散状

#### （四）变形场锋生型

| 项目    | 内容                                                              |
| ----- | --------------------------------------------------------------- |
| 冷空气源地 | 冰岛以南的洋面                                                         |
| 冷空气路径 | 西方路径（欧洲南部→里海→巴尔喀什湖→新疆→我国）                                       |
| 关键区入侵 | 河西走廊西路冷空气（西路）+ 华北和东北回流冷空气（东路）汇合                                 |
| 环流特征  | 500hPa东亚倒Ω流型形成后，极涡南下到贝湖，亚洲北部形成纬向低压带；低层（850/700hPa）在河套形成鞍形场（变形场） |
| 爆发方式  | 两股冷空气在河套汇合形成锢囚锋缓慢南下，持续降温                                        |
| 东亚大槽  | 欧亚大陆长波环流形势**稳定**（非调整）                                           |

**鞍形气压场**（变形场）：两个高气压和两个低气压交错相对的中间区域。_变形场有利于锋生。_

#### （五）横槽旋转型

| 项目    | 内容                             |
| ----- | ------------------------------ |
| 冷空气源地 | 新地岛以东的洋面                       |
| 冷空气路径 | 北方路径                           |
| 环流特征  | 东亚倒Ω流型形成，极涡在西伯利亚为强大冷低中心，向西伸出横槽 |
| 爆发方式  | 横槽沿极涡中心旋转运动，带下一股股冷空气，持续降温影响北方  |
| 东亚大槽  | 欧亚大陆长波环流形势**稳定**（非调整）          |

## 3.2 降水的形成条件

### 一般降水条件

1. **水汽条件**——水汽从源地水平输送到降水区
2. **垂直运动条件**——降水区的辐合上升
3. **水滴增长过程**——冰晶效应、碰撞合并

云滴增长取决于云层厚度，云层厚度取决于水汽条件和垂直运动条件。降水分析就是对水汽条件和垂直运动条件进行分析。

### 暴雨形成的三个条件（_必背_）

1. ==**充分的水汽供应**==：整层大气水汽含量高，且有足够水汽从源地不断向该地区输送
2. ==**强烈的上升运动**==：暴雨是中小尺度系统的产物，需要有很强的不稳定能量储存和释放
3. ==**较长的持续时间**==：大尺度环流系统稳定、天气尺度系统重复出现、中小尺度系统发展活跃

## 3.6 各区域降水特点

### 3.6.1 华南前汛期（4-6月）

**特征**：5月中旬-6月上旬为盛期，暴雨多。**锋前暖区暴雨**是华南特色——降水比锋面降水大3-5倍，局地性强。

**环流类型（500hPa）**：

1. **两脊一槽型**：贝湖脊+鄂海脊，中间为低槽
2. **两槽一脊型**：贝湖槽+鄂海槽，中间为高脊
3. **多波动型**：中纬多短波槽脊，无明显长波调整

**共同特征**：中高纬长波槽脊稳定；副高脊线稳定在15-18°N；100hPa南亚高压控制中南半岛（高层辐散）。

**暖区暴雨触发机制**：边界层浅薄冷空气侵入、地形抬升、海陆风辐合。

### 3.6.2 江淮梅雨（6月中旬-7月上旬）

**定义**：夏初长江中下游的连阴雨，发生在宜昌以东28-34°N。

**环流形势（垂直四层结构，高频考点）**：

| 层次  | 气压         | 特征                                                                  |
| --- | ---------- | ------------------------------------------------------------------- |
| 高层  | 100-200hPa | 南亚高压东移稳定在27°N，110°E                                                 |
| 中层  | 500hPa     | 副高脊线第一次北跳至22°N（入梅），第二次北跳至25-30°N（出梅）；中纬度平直西风多短波槽；高纬有阻塞高压（单阻、双阻、三阻型） |
| 低层  | 850/700hPa | **江淮切变线**和**西南低空急流**                                                |
| 地面  | —          | 江淮静止锋、气旋波                                                           |

**梅雨锋特征**：**湿度对比明显**（比湿梯度大），温度梯度时有时无（因北方变性高压下沉增温抵消冷平流）。

**暴雨系统**：500hPa短波槽（弱冷空气）、700/850hPa切变线上低涡（西南涡）和低空急流（水汽输送）。

**入梅/出梅**：副高脊线第一次北跳至22°N为入梅，第二次北跳至25-30°N为出梅。

### 3.6.3 华北与东北雨季（7-8月，"七下八上"）

**特征**：降水强度大、持续时间短、局地性强、年际变化大。与地形关系密切（迎风坡）。

**华北暴雨环流类型**：

1. **两高对峙**：西太平洋副高+大陆高压对峙
2. **三高对峙**：西太平洋副高+青藏高压+华北暖高压对峙
3. **东高西低**：日本海副热带高压阻挡上游系统东移，输送暖湿空气

关键系统：**日本海副热带高压**（阻挡作用、暖湿输送）。

**东北暴雨环流**：500hPa低槽位于110-120°E，中低层西南风急流，地面气旋（黄河气旋、江淮气旋）移入。

## 3.7 降水的天气尺度系统

### 3.7.1 低空切变线

**定义**：低空（850hPa、700hPa）风场上具有气旋式切变的不连续线。

**四种类型**：

| 类型    | 出现季节 |
|:-----:|:----:|
| 华南切变线 | 春季   |
| 江淮切变线 | 6-7月 |
| 华北切变线 | 7-8月 |
| 高原切变线 | 夏季   |

**江淮切变线的分类**：

| 类型          | 特征                               | 速记     |
|:-----------:|:--------------------------------:|:------:|
| 冷锋式切变（冷式切变） | 偏北风/西北风与西南风的切变，偏北风占主导，自北向南移动     | 冷峰：北>南 |
| 暖锋式切变（暖式切变） | 东南风/偏东风与西南风/偏南风的切变，偏南风占主导，自南向北移动 | 暖锋：北<南 |
| 静止锋式切变      | 偏东风与偏西风的切变，南北两支气流势力相当，很少移动       | 静止：北≈南 |

**江淮切变线的移动**：随副热带高压南北位移；冷空气强则南移，暖空气强则北移；低涡沿切变线东移时，切变线南北摆动一次。

**江淮切变线的形成**：南北向的西风带短波槽在带状副高北侧顺转成东西向切变线。

### 3.7.2 西南低涡（西南涡、必背）

**定义**：形成于四川西部700（850）hPa的闭合小低压，直径300-400km。

**形成原因**（4个）：

1. 高原地形——背风坡动力作用
2. 500hPa高原槽东移诱生
3. 700hPa西南气流辐合
4. 江淮切变线西端

**发展条件**：

- 500hPa有发展槽配合（北槽南涡）
- 冷空气从西部或西北部侵入（东或东北部侵入则不发展）
- 移出源地后才发展

**移动路径**：偏东、东北、东南三路。移向与500hPa风向基本一致，移速为500hPa风速的50-70%。

**天气**：移出源地的西南涡降雨量大，雨区在涡中心和东南侧，夜间雨大。

### 3.7.3 高空冷涡（东北冷涡）

**定义**：东北地区附近有闭合等高线（≥2根），维持3-4天，有深厚冷空气。

**季节**：5-6月最多。

**天气**：低温、不稳定降水（雷阵雨），午后到前半夜降水大。

### 3.7.4 低空急流（西南低空急流）

**定义**：850/700hPa风速≥12m/s的西南风极大风速带。

**与暴雨关系（必背）**：

- 暴雨区出现在急流中心左侧200km以内
- 79%的低空急流伴有暴雨，83%的暴雨伴有低空急流
- 急流宽度越宽，暴雨强度越大
- 夏季，急流与高空急流入口区右侧、南亚高压东部脊线配合，形成高层辐散、低层辐合

**形成与维持**：高空辐散（西风急流或南亚高压）导致低层减压，气压梯度加大，偏南风加快；暴雨区潜热加热进一步降压，形成正反馈。

## 3.8 水汽诊断关键量

| 变量     | 含义                         |
|:------:|:--------------------------:|
| 比湿（q）  | 出现暴雨时，850hPa比湿通常≥8-10g/kg  |
| 可降水量   | 单位面积整层大气水汽全部凝结的降水量         |
| 水汽通量   | 单位时间通过单位面积的水汽量             |
| 水汽通量散度 | 表示水汽水平辐合辐散。降水区主要由低层风水平辐合造成 |

**垂直运动诊断**：低层辐合+高层辐散→上升运动（暴雨）。高层（200-300hPa）槽前正涡度平流→高层辐散→上升运动。

五种基本线条：

- 等温线（与纵坐标平行）
- 等压线/等高线（与横坐标平行）

# 第四章 青藏高原天气

本章讲的是青藏高原的独特天气气候特征及其对中国天气的影响。考试重点在"五个度"、平均环流（冬/夏）、降水特征，以及高原特有系统（切变线、低涡、南亚高压）。

## 4.1 青藏高原地理特征——"五个度"

| 特征  | 描述                    |
|:---:|:---------------------:|
| 尺度  | 250万平方公里              |
| 纬度  | 北半球副热带                |
| 经度  | 欧亚大陆东部                |
| 高度  | 接近对流层中部（平均4000-5000m） |
| 坡度  | 东南边缘陆陆地形              |

南北最宽约1000公里，东西最长达2000公里，是世界上面积最大、海拔最高的高原，有"世界屋脊"之称。

## 4.2 平均环流

### 冬季（5000m高度）

- 南北气流分支，与地形等高线一致
- **新疆脊与孟加拉湾槽**——构成北脊南槽
- 高原西部辐散、东部辐合
- 南北各有一支西风急流

3000m：流线外形与5000m相近，但东亚大槽位置更偏东。高原主体东侧南北各有一个小涡旋。

1500m：高原以北的新疆至河西走廊有一连串闭合的反气旋环流；高原以南，气流沿地形爬升。

### 夏季（500hPa）

- 西风带北退，高原主体为平均的副热带高压
- 高原以东为西太副高西伸脊，以西为伊朗高压东伸脊
- 新疆长波脊和东亚大槽比冬季短
- 西风带和季风气流直接汇合形成500hPa切变线，常有次天气尺度500hPa低涡活动

3000m以下：东部为西南季风与西风气流交汇形成切变线；西南季风越过横断山脉后在川西背风坡形成动力性气旋涡——**西南涡**。

## 4.3 平均压、温、湿场

### 气压场

- _夏季是热低压，冬季是冷高压_
- **地面气压年变化与平原相反**：夏季气压高、冬季气压低
- 高原地面气压日变化振幅比平原大

### 温度场

- 与同纬度相比，高原地面年平均气温比四周低10-14℃
- 与自由大气相比，**夏季高原比同纬度平原气温高5-7℃**，成为叠加在中纬度的独立暖中心（可达200hPa以上）
- 气温年较差比平原小，**日较差比平原大**
- 有人称青藏高原为"世界第三冷极"

### 湿度场

- 同纬度相比：全年平均绝对湿度只相当于同纬度平原的1/3，年平均相对湿度40-50%（平原60-80%）
- 同高度自由大气相比：夏季高原上是高湿区，比湿比印度、巴基斯坦还大

> [!TIP]
> 青藏高原_夏季是热源，冬季是冷源_
>
> - **夏季（热源）**：地表净辐射为正，通过强烈的**感热**和**凝结潜热**加热对流层中低层，形成深厚的**热力性低压**，驱动周围气流辐合上升。
> - **冬季（冷源）**：净辐射为负，叠加高反照率积雪的冷却效应，形成**冷性高压**，盛行下沉辐散气流。
> - **转换期**：冷暖源转换通常发生在**3—4月**（冷转热）和**10月**（热转冷）。
> - **气候效应**：该季节性反转是东亚季风爆发及北半球副热带环流调整的关键下垫面强迫因子。

## 4.4 垂直环流

### 经向环流

- 夏季：青藏高原是一个巨大的**季风环流圈**
- 冬季：中低纬上空是强大的Hadley环流圈，北侧为逆环流

### 纬向环流

- 冬季：高原处于西风带，东西两侧各有一个波谷
- 夏季：高原范围内存在一个巨大的上升气柱，一直上升到200hPa以上，东至180°E以东才下沉

**重要结论**（叶笃正）：夏季青藏高原加热引起的热力环流主要影响东太平洋高压，而对西太平洋高压作用不大。

## 4.5 降水特征

**地理分布**：==自东南向西北减少==。藏东南600-800mm，藏西北与新疆交界处50mm以下。我国降水最少地区：柴达木西北部（年均17.6mm）。我国第二多雨中心：藏南雅鲁藏布下游（昔卡年均超4400mm）。

**季节变化**：降水主要在夏季（5-9月占全年80-90%），雨季和干季分明。喜马拉雅山南麓和雅鲁藏布江下游为双峰型。

**日变化**：夜雨多于白昼。藏南夜雨率>60%，雅鲁藏布河谷夜雨率高达80%以上。

**地形影响**：降水量在一定高度范围内随海拔增高而增加，到达最大降水高度（约3600m）后随海拔增高而减少。

## 4.6 高原切变线和低涡

### 高原500hPa切变线

夏季准定常系统，分两类：

| 类型            | 特征                      |
|:-------------:|:-----------------------:|
| **横切变线**（东西向） | 6-7月最多，平均位置32.5°N，西段略南移 |
| **竖切变线**（南北向） | 8月最多，两条高频带（高原陡坡和中部）     |

切变线浅薄（不及400hPa），有明显湿度梯度。南移后常给川滇带来暴雨。

### 高原低涡（500hPa）

| 项目    | 内容                                   |
|:-----:|:------------------------------------:|
| 热力性质  | 大部分为**暖性结构**（2/3），是高原独有系统            |
| 源地    | 高原中、东部（90°E以西）                       |
| 移动    | 多数消失于高原东坡（下坡处），与一般低压在下坡加强相反，是特殊热力产物  |
| 移出率   | 仅约24%移出高原                            |
| 移出后影响 | 移出后在四川及以东产生大暴雨                       |
| 结构    | 水平尺度400-600km，暖心偏于西南，南部为西南季风，北部为反气旋脊 |

## 4.7 南亚高压（青藏高压）

**定义**：夏季对流层高层（100hPa附近）的庞大暖性高压，是最强最稳定的系统之一。

**结构特征**：

- 行星尺度反气旋环流
- **暖心结构**，内部为上升气流（不同于太平洋副高的下沉），对流活动活跃
- 属于**热力性质**的高压（热力作用为主），与太平洋副高（动力为主）不同

**活动与天气影响**：

- **季节性南北位移**：5月到中南半岛（23°N），6月到青藏高原（28°N），7-8月到最北（32-33°N），9月东移入海
- **东西振荡**：
  - **西部型**（中心90°E以西）→长江流域多雨
  - **东部型**（中心90°E以东）→长江流域少雨
- 脊线四次北跳比西太平洋副高提前约10天，对应华南前汛期、江淮梅雨、华北雨季等

## 4.8 高原对东亚天气系统的影响

| 作用类型     | 具体影响                                           |
|:--------:|:----------------------------------------------:|
| **动力作用** | 阻挡、切断长波槽；西风分支成南北两支，形成"北脊南槽"；东侧形成江淮切变线、西南涡等特殊系统 |
| **热力作用** | 夏季热源维持南亚高压，使500hPa副高断裂；冬季冷源加强西伯利亚高压            |
| **对寒潮**  | 阻挡冷空气南下，使冷空气沿高原北侧堆积，从东部边缘加速南下（对印度形成屏障）         |
| **对降水**  | 高原东坡和南坡是降水多发区，季风沿高原爬升                          |

## 4.9 本章速记（考试必背）

1. "五个度"：尺度、纬度、经度、高度、坡度
2. 冬季环流：北脊南槽（新疆脊+孟湾槽），南北分支气流
3. 夏季环流：高原为副高主体，西风季风交汇形成切变线
4. 地面气压年变化与平原相反：夏高冬低
5. 夏季高原比同纬度平原气温高5-7℃，是独立暖中心
6. 自东南向西北降水递减，夜雨率>60%
7. 南亚高压是热力性质高压（与太平洋副高的动力性质不同），东西振荡影响长江流域旱涝
8. 高原切变线：横切变线（东西向，6-7月）和竖切变线（南北向，8月）
9. 高原低涡：2/3暖性，仅24%移出高原，移出后产生大暴雨
10. 夏季高原是巨大季风环流圈；高原加热影响东太平洋高压

# 第五章 热带环流系统与东亚季风

本章讲的是低纬度（30°N-30°S）的大气运动规律和主要环流系统。考试重点在低纬与中高纬运动差异、赤道辐合带（ITCZ）、太平洋副热带高压、ENSO、台风以及东亚季风系统。

## 5.1 低纬度大气运动的基本特征

### 5.1.1 特征尺度

天气学中区分热带和副热带的依据：

- **热带地区**：南北半球副热带高压之间的东风带控制区
- **副热带地区**：低纬东风带和中纬西风带之间的过渡区

| 纬度带       | 准地转关系 |
|:---------:|:-----:|
| 低纬度天气尺度系统 | 非地转   |
| 低纬度行星尺度系统 | 准地转   |
| 中高纬度系统    | 准地转   |

**科里奥利参数 f 很小**，气压场水平梯度比中高纬小，流场水平差异却十分明显。热带天气系统发生往往先出现流场涡旋，气压场特征在产生强烈对流后才逐渐明显。

### 5.1.2 垂直速度和散度

| 条件         | 垂直速度 ω      | 垂直速度 W     | 散度 ∇V |
|:----------:|:-----------:|:----------:|:-----:|
| 无降水        | ~10⁻⁴ hPa/s | ~10⁻¹ cm/s | ~10⁻⁷ |
| 有降水（2cm/日） | ~10⁻³ hPa/s | ~10⁰ cm/s  | ~10⁻⁶ |

**低纬垂直速度特点**：

- 降水区垂直速度比无降水区大一个量级
- 无降水区大气运动为准水平
- 垂直运动主要由非绝热加热引起——热带大气水汽充沛，凝结潜热效应显著

**低纬散度特点**：

- 降水区散度比无降水区大一个量级
- 无降水区为准无辐散大气运动

### 5.1.3 扰动的主要能源

- **中高纬度**：斜压不稳定为扰动发展的主要能源
- **低纬度降水区**：水汽凝结潜热是扰动发展的主要能源
- **低纬度无降水区**：正压不稳定为扰动发展的主要能源

### 5.1.4 风场与气压场的关系

**20°N作为分界线**：

- 20°N以北：实际风与地转风较一致，准地转
- 20°N以南：实际风与地转风偏差较大，非地转

**压能风**：热带地区风场与压能场关系类似于中高纬地转风平衡的关系。

- 压能风风向与等Φ线平行，等Φ线近似热带流线
- 风速大小与|∇Φ|成正比，与η（绝对涡度）成反比
- η=0 为动赤道

### 5.1.5 中低纬运动的重要差别

1. 低纬度 f 很小，不能满足地转风关系
2. 热带大气中凝结潜热效应显著影响垂直速度和水平散度

## 5.2 低纬度环流的基本特征

### 5.2.1 海平面气压场和流场

**气压场特征**：

1. 南北半球海洋上均为常年存在的副热带高压，夏强冬弱（太平洋副高和大西洋副高）
2. 大陆上冬季为冷性高压，夏季为暖性低压
3. 赤道附近为赤道低压带

**流场特征——赤道辐合带（ITCZ）**：
南北两支信风交汇地带，随季节南北移动。

| 区域              | 类型         | 特征                          |
|:---------------:|:----------:|:---------------------------:|
| 太平洋中部、美洲、大西洋    | 信风辐合带（信风槽） | 常年位于20-30°N，吹东风，扰动弱         |
| 非洲-印度洋-南亚-太平洋西部 | 季风辐合带（季风槽） | 1月10-20°S，7月20-30°N，静风，扰动活跃 |
| 赤道附近            | 赤道低压带      | 季风越赤气流显著                    |

**ITCZ结构**：低层辐合，高层辐散，随高度向南倾斜，两侧温差小（<3℃），湿度南高北低。

**与台风关系**：_西太平洋80%台风形成于ITCZ中_。

### 5.2.2 对流层高层（200hPa）气压场和流场

**两种急流带**：

1. 南北半球各有一支副热带西风急流（冬强夏弱，南北位移）。冬季北半球三个副热带急流中心：地中海、日本沿海、美国东南部
2. 北半球夏季北非-南亚-西太平洋存在热带东风急流
3. 亚洲低纬地区，冬季副热带西风急流和夏季东风急流均为全球最强

**夏季北半球低纬主要长波系统**：

- 三个高压中心：**南亚高压**、西非高压、墨西哥高压
- 两个长波槽：太平洋热带对流层高层槽、大西洋热带对流层高层槽

**冬季北半球低纬主要长波系统**：太平洋中部、大西洋中部以及印度洋上空呈现为洋中槽（TUTT）。

### 5.2.3 平均温度场和湿度场

**海平面温度**：热带气温年较差小（全年均温），陆上年较差大，海洋上年较差小。

**300hPa温度**：

- 冬季：赤道附近三个暖中心（印度尼西亚、南美西北部、赤道非洲）→对应三个热带对流降水区
- 夏季：南亚高压、西非高压、墨西哥高压均为暖中心区，大洋中部槽为冷区

**850hPa湿度**：夏季季风区为比湿最大值区。

### 5.2.4 经向垂直环流——Hadley环流

- 直接热力环流：赤道为上升支，副热带为下沉支
- 上升支季节性位移：北半球冬季位于5°S，夏季位于10°N

**经向环流的纬向不对称性**：

- 东太平洋（信风区）→三圈环流
- 西太平洋（信风与季风过渡区）→冬季Hadley环流，夏季Hadley反环流
- 南亚季风区→冬季Hadley环流，夏季季风环流圈（Hadley反环流圈）

### 5.2.5 纬向垂直环流——沃克环流（Walker Circulation）

**定义**：在低纬地区西太平洋暖水区上升，至东太平洋冷水区下沉，所构成的纬向环流圈。与南方涛动（SO）有密切关系。

**上升支位置**：1月三个（印度尼西亚、赤道非洲、南美西北部）；7月两个（亚洲季风区、北美墨西哥南部）。

## 5.3 ENSO（厄尔尼诺/南方涛动）

**厄尔尼诺（El Nino）**：赤道中东太平洋海温持续偏高并造成大气环流异常的气候现象（"圣婴"）。

**拉尼娜（La Nina）**：赤道中东太平洋海温大范围持续异常偏冷（"小女孩"）。

**周期**：2-7年不规则周期，分厄尔尼诺、拉尼娜和中性三个阶段。

### 厄尔尼诺对我国天气气候的影响

- **台风减少**（西北太平洋热带风暴及登陆次数减少）
- **北方夏季易高温干旱**（夏季风弱，雨带偏南）
- **南方易低温洪涝**（次年长江流域和江南易现洪涝）
- 厄尔尼诺发生后的冬季，北方易出现暖冬

### 拉尼娜对我国天气气候的影响

- 当年秋季北方降水偏多，出现秋汛可能性较大
- 当年冬季中东部气温偏低，出现冷冬可能性较大

## 5.4 太平洋副热带高压

**定义**：形成和活动于太平洋副热带地区的暖性高压。常年存在，稳定少动的深厚暖性系统。夏季强大范围广，冬季较弱范围小。对我国影响最大的是**西太平洋副高**。

**结构**：暖区，脊中干燥；脊线随高度变化（夏季低层向北倾斜，高层垂直或向南倾斜）；低层辐散，高层辐合；脊轴附近有反环流。

**季节变动与我国天气（必背）**：

| 时间          | 副高脊线位置  | 天气影响               |
|:-----------:|:-------:|:------------------:|
| 4-5月        | 15°N    | 华南雨季               |
| 6月中旬（第一次北跳） | 22°N    | 江淮梅雨               |
| 7月中旬（第二次北跳） | 28°N    | 梅雨结束，雨带北移至黄淮，华南后汛期 |
| 8月初         | 30-35°N | 华北东北雨季，长江流域伏旱      |
| 9月          | 南撤      | 华西秋雨               |

**与周围系统的相互作用**：西风带强脊使副高西伸北进，深槽使其东撤南退；大陆高压（青藏高压、华北暖高压）并入可使副高加强西伸；台风可穿过副高脊使其断裂。

## 5.5 台风（热带气旋）

**定义**：生成于热带/副热带海洋的**暖心、无锋**的气旋性涡旋，伴有深对流和闭合环流。

### 强度分类（我国标准）

| 等级    | 风速                   |
|:-----:|:--------------------:|
| 热带低压  | 6-7级（10.8-17.1m/s）   |
| 热带风暴  | 8-9级（17.2-24.4m/s）   |
| 强热带风暴 | 10-11级（24.5-32.6m/s） |
| 台风    | 12-13级（32.7-41.4m/s） |
| 强台风   | 14-15级（41.5-50.9m/s） |
| 超强台风  | ≥16级（≥51.0m/s）       |

### 台风结构

- **眼区**：晴空无风，气压最低
- **眼墙**：最大风速、最强降水
- **螺旋雨带**：外围雨带

**环流特征**：切向风在眼墙处最大；径向环流为低层流入、高层流出；暖心结构在250hPa最暖。

### 台风的形成条件（5个，必背）

1. 初始扰动（启动机制）
2. 低层绝对涡度足够大（远离赤道5°以上）
3. 暖洋面（SST≥26.5℃，暖水层深）
4. **弱的水平风垂直切变**（<10m/s有利）
5. 对流层中层相对湿度大

### 移动路径

| 路径          | 影响地区      |
| ----------- | --------- |
| **西行**      | 进入南海，影响华南 |
| **西北移**     | 影响台湾、华东   |
| **转向（抛物线）** | 转向日本或朝鲜半岛 |

**引导气流**：主要受副热带高压外围气流引导。台风在副高南侧西行，在西侧转向北上。

## 5.6 东亚季风环流系统

**季风定义三要素**：

1. 盛行风冬季相反
2. 气团性质根本不同（冬干冷，夏暖湿）
3. 造成季节性天气差异（雨季/旱季）

**形成原因（4个）**：

1. **海陆热力对比**——基本原因
2. **行星风带季节移动**——低纬度显著
3. **青藏高原大地形**——加强季风，维持南亚高压和东风急流
4. **南北半球气流相互作用**——越赤道气流

**东亚夏季风系统成员（高低空配合）**：

- **低空**：澳大利亚冷性反气旋→越赤道气流（110°E附近）→南海季风槽（ITCZ）→西南季风；西太平洋副高南侧的东南季风
- **高空**：南亚高压东部脊、热带东风急流
- **交汇处**：梅雨锋（副热带季风辐合带）

东亚季风是全球最显著的季风区，因其海陆对比和青藏高原共同作用。

## 5.7 本章速记（考试必背）

1. 低纬与中高纬本质差异：f很小→非地转；凝结潜热起主导作用
2. 20°N是地转适用分界线，以北准地转，以南非地转
3. 能源：中高纬→斜压不稳定；低纬降水区→凝结潜热；低纬无降水区→正压不稳定
4. ITCZ分信风槽（扰动弱）和季风槽（扰动活跃），80%台风在ITCZ中形成
5. 沃克环流=西太平洋暖水区上升→东太平洋冷水区下沉的纬向环流
6. ENSO周期2-7年：厄尔尼诺（暖）→台风少、北方热干、南方涝；拉尼娜（冷）→北方秋汛、冬季冷冬
7. 副高脊线决定雨带：5月15°N（华南）、6月22°N（入梅）、7月28°N（出梅）、8月30-35°N（华北雨季）
8. 台风5形成条件：初始扰动、远离赤道、SST≥26.5℃、弱垂直切变、中层高湿
9. 台风3路径：西行、西北移、转向（抛物线）
10. 东亚季风形成四原因：海陆对比、行星风带移动、青藏高原、越赤道气流

16. **南亚高压（青藏高压）**：夏季100hPa附近的热力性暖高压
17. **沃克环流**：赤道太平洋纬向垂直环流圈，暖水区上升、冷水区下沉
18. **ENSO**：厄尔尼诺+南方涛动，2-7年周期
19. **台风（热带气旋）**：热带海洋上的暖心无锋气旋性涡旋
20. **东亚季风**：全球最显著季风区，海陆+青藏高原共同作用
